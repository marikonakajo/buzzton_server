# buzzton_server

buzztone server

## how to start server

`docker-compose build`
`docker-compose up`

execute container as background process

`docker-compose up -d`

## connect to container 

`docker-compose ps`

## login to node container

`docker exec -it nodeapp /bin/sh`

## stop & start container

`docker-compose stop`
`docker-compose start`
